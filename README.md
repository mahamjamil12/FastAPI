# FastAPI Blog API

A simple RESTful Blog API built with **FastAPI**, **SQLAlchemy**, and **SQLite**.  
This project demonstrates CRUD operations, database integration, and secure user creation with password hashing.

---

## 🚀 Features

- FastAPI framework
- SQLite database
- SQLAlchemy ORM
- CRUD operations for blogs
- User creation with hashed passwords (bcrypt)
- Dependency injection for database sessions
- Automatic API documentation (Swagger UI)
